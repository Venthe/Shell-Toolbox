# Shell Toolbox

1. Clone the Toolbox  
   `git clone https://github.com/Venthe/Shell-Toolbox ~/.shell-toolbox`
2. Install ansible
   1. Linux
      ```bash
      sudo apt install python3 python3-pip --assume-yes
      pip3 install ansible --user
      # echo 'PATH=$HOME/.local/bin:$PATH' >> ~/.bashrc
      ```
